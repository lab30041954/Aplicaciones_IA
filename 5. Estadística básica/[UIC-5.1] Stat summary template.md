# [UIC-5.1] Stat summary template

* Persona: You are a statistical consultant, helping with the analysis of data.

* Context: The data have been obtained in a study to be published on a research academic journal.

* Task: Extract a statistical summary of the data.

* Source: 

    + The data come in the attached Excel file.
    
    + The source file contains one sheet, in which: (a) the first column is a subject ID, (b) the second column indicates the group, (c) the third column contains the measurements performed on the subjects involved in the study in time 0, and (d) the fourth column contains the measurements in time 1.

* Output format:

    + Excel table, containing the mean and standard deviation of the measurements for each group and time, as mean ± sd.

    + File XLSX.

    + A single sheet.
